# AI Recorder UX Flow Review

```mermaid
flowchart LR
    A["AI Key<br/>一键启动录音"] --> B["前台录音体验<br/>大折叠左右两栏"]
    B --> C["后台/息屏<br/>玲珑台展示录音进程"]
    C --> D["AI Key<br/>一键停止录音"]
    D --> E["玲珑台完成态<br/>点击回到录音详情"]
    E --> F["录音详情<br/>Transcription / Summary / Chapter / Speakers"]
    F --> G["生成摘要<br/>会议模板 · 常规颗粒度 · 默认在线模型"]
    G --> H["摘要结果<br/>场景切换 · 思维导图 · 时间戳回跳"]
    H --> I["行动项<br/>一键创建 Reminder"]
    H --> J["底部提问栏<br/>当前文件问答 / 跨文件问答"]
    J --> K["项目复盘文档<br/>导出 / 邮件 / 微信 / Teams / 飞书 / 钉钉"]

    subgraph Fold["大折叠前台结构"]
        L["左栏<br/>All Recordings<br/>录音列表/状态"]
        R["右栏<br/>当前录音详情<br/>实时转写/摘要/章节/说话人"]
        L --- R
    end

    B -. 对应 .-> Fold
```
